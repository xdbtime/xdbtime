Database Performance Comparison Report

for SQL Server (based on Query Store data) v2025.12

DB Time charts are based on query_store_runtime_stats view (CPU Time) and query_store_wait_stats view (Buffer IO and Other wait time)

Wait Classes charts are based on query_store_wait_stats view

SQL Statistics charts are based on query_store_runtime_stats view. Query statistics are grouped by query_id and execution plan. SQL classification is based on query. There are 3 default classes: Aged Out, New and Unclassified.

Query statistics tables are based on query_store_runtime_stats view. Query statistics are grouped by query_id and execution plan. SQL classification is based on query. There are 3 default classes: Aged Out, New and Unclassified.

Query text is based on query_store_query_text
